## 🚀 What You Need to Do (2 minutes)

### Step 1: Enable GitHub Pages

1. Go to: https://github.com/nbedzeti/ToneForge/settings/pages
2. Under **"Build and deployment"**:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/docs`
3. Click **Save**

### Step 2: Wait 1-2 Minutes

GitHub will build and deploy your site automatically.

### Step 3: Verify It Works

Visit: https://nbedzeti.github.io/ToneForge/privacy.html

You should see your privacy policy page.
