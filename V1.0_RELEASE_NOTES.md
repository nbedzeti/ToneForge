# ToneForge Studio v1.0 - Release Notes

## Version 1.0 - Initial App Store Release

**Release Date:** January 2026  
**Build:** 1

## What's Included in v1.0

### Core Features
✅ **Audio Selection**
- Import from Music Library
- Import from Files app
- Support for MP3, M4A, WAV, AAC formats

✅ **Audio Editing**
- Visual waveform display
- Precise time selection (up to 30 seconds)
- Real-time preview
- Custom number pad for time input

✅ **Premium Audio Effects** (Premium Only)
- Fade in/out
- Volume boost
- Audio normalization
- Reverb
- Echo
- 5-band equalizer

✅ **Ringtone Library** (Premium Only)
- Save ringtones to library
- Organize and manage saved ringtones
- Quick access to favorites

✅ **Monetization Model**
- 3 free ringtone creations
- Premium subscriptions:
  - Monthly: $2.99/month
  - Yearly: $17.99/year (50% savings)
  - Lifetime: $24.99 (one-time purchase)

---

## What's NOT in v1.0 (Coming in v1.1)

### Ad Functionality - Deferred to v1.1
❌ **Rewarded Video Ads**
- Watch 3 ads = unlock 1 ringtone
- Ad progress tracking
- AdMob integration

**Why deferred?**
- AdMob SDK requires CocoaPods setup
- CocoaPods not installed in current environment
- Decided to ship v1.0 with simpler monetization model
- Will add ads in v1.1 update after proper SDK integration

## Code Changes for v1.0

### Files Modified

1. **PurchaseManager.swift**
   - Commented out ad-related properties
   - Simplified `canCreateRingtone` logic
   - Disabled `AdManager` initialization
   - Commented out `showRewardedAd()` and `rewardedAdWatched()` methods
   - Updated `UsageStats` struct

2. **PurchaseView.swift**
   - Removed "Watch Ad" button
   - Removed ad progress tracking UI
   - Removed ad-unlocked ringtones display
   - Updated features list (removed "No Ads")
   - Simplified upgrade flow

3. **ContentView.swift**
   - No changes needed (already shows PurchaseView when limit reached)

4. **AdManager.swift**
   - No changes needed (file exists but not used in v1.0)
   - Will be activated in v1.1
