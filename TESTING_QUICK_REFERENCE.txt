╔══════════════════════════════════════════════════════════════════════════════╗
║                         FIREBASE ANALYTICS                                   ║
╚══════════════════════════════════════════════════════════════════════════════╝

┌─ SETUP (30 MINUTES) ─────────────────────────────────────────────────────────┐
│                                                                              │
│  1. Create Firebase project at console.firebase.google.com                   │
│  2. Add iOS app with bundle ID                                               │
│  3. Download GoogleService-Info.plist                                        │
│  4. Add to Xcode project                                                     │
│  5. Install Firebase SDK via SPM:                                            │
│     https://github.com/firebase/firebase-ios-sdk                             │
│  6. Initialize in RingToneMakerApp.swift:                                    │
│     FirebaseApp.configure()                                                  │
│  7. Add AnalyticsManager.swift (see guide)                                   │
│  8. Integrate events in key flows                                            │
│  9. Enable DebugView: -FIRDebugEnabled                                       │
│  10. Test events in Firebase Console                                         │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ KEY EVENTS TO TRACK ────────────────────────────────────────────────────────┐
│                                                                              │
│  FREE TIER:                                                                  │
│    • free_creation_used (remaining, total_used)                              │
│    • free_limit_reached (time_since_install)                                 │
│                                                                              │
│  REWARDED ADS:                                                               │
│    • ad_requested (trigger)                                                  │
│    • ad_loaded (load_time)                                                   │
│    • ad_impression (ad_unit_id)                                              │
│    • ad_reward_earned (ads_watched, unlocked)                                │
│    • ad_failed (error)                                                       │
│                                                                              │
│  PURCHASES:                                                                  │
│    • paywall_viewed (trigger)                                                │
│    • product_selected (product_id, price)                                    │
│    • begin_checkout (product_id, value)                                      │
│    • purchase (product_id, value, transaction_id)                            │
│    • purchase_failed (product_id, error)                                     │
│    • restore_purchases (success)                                             │
│                                                                              │
│  CONVERSIONS:                                                                │
│    • conversion_free_to_ad (time_since_limit)                                │
│    • conversion_free_to_premium (product_id, time_since_install)             │
│    • conversion_ad_to_premium (product_id, total_ads_watched)                │
